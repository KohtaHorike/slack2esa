

# slack2esa
保存したいslackのスレッドにesaスタンプを付与すると、esaに保存を行うツール。

Slack内で、esaスタンプ（任意のスタンプでOK）を設定し、保存したいslackスレッドにスタンプを付与することでGoogle App Script経由でesaに投稿されます。
